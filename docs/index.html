Blackboard Bookmarklets

This project contains a collection of experimental bookmarklets for use with Blackboard. These tools are designed to streamline certain administrative tasks and improve workflows.

All code is open source and available on GitHub. You can view the source or contribute here:

View Repository on GitHub
To install: Drag one of the buttons below to your bookmarks bar. Then, click it while on a Blackboard page to run the tool.
